# 🚗 JS Mini Projects

A collection of **5 beginner-friendly JavaScript projects** built with pure HTML, CSS, and vanilla JavaScript — no frameworks, no libraries.

---

## 🔴 Live Demo

Open `index.html` in any browser — no setup, no install needed.

---

## 📁 File Structure

```
js-mini-projects/
├── index.html     # Main HTML file (structure + content)
├── style.css      # All styles and layout
├── script.js      # All JavaScript logic
└── README.md      # Project documentation
```

---

## ✨ Projects Included

| # | Project | Description |
|---|---------|-------------|
| 01 | **Background Color Changer** | Click a button to generate a random background color, or pick from preset color swatches |
| 02 | **Live Text Preview** | Type in an input field and see the text mirror instantly in a preview box with live character count |
| 03 | **Show / Hide Password** | Toggle password visibility with an eye icon, plus a live strength meter (weak → very strong) |
| 04 | **Car Image Slider** | Auto-playing slider with 5 real car photos (Lamborghini, Porsche, Ferrari, BMW, Mercedes), Prev/Next buttons, and dot navigation |
| 05 | **Counter** | Increment or decrement a counter with a custom step value; number turns green/red based on value |

---

## 🛠️ Concepts Covered

- DOM manipulation (`getElementById`, `textContent`, `innerHTML`)
- Event handling (`onclick`, `oninput`)
- CSS class toggling (`classList.add`, `classList.remove`)
- Dynamic element creation (`createElement`, `appendChild`)
- CSS transitions and animations
- `setInterval` for auto-play
- `Math.random()` for random colors
- Password strength logic using regex
- `input.type` toggle for show/hide password

---

## 🚀 How to Run

1. **Clone the repo**
   ```bash
   git clone https://github.com/your-username/js-mini-projects.git
   ```

2. **Open the project folder**
   ```bash
   cd js-mini-projects
   ```

3. **Open in browser**
   ```bash
   open index.html
   ```
   Or simply double-click `index.html` — it runs straight in the browser.

> ⚠️ An internet connection is required to load the car images (hosted on Unsplash).
